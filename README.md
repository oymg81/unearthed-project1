# WEB103 Project 1 - Unearthed Gift Ideas

Submitted by: Oscar Mochizaki

About this web app: **Unearthed Gift Ideas is a list-based web app that displays unique and thoughtful gift ideas. Each gift includes a name, category, price, description, and image. The app is built with vanilla HTML, CSS, JavaScript, Express, and PicoCSS.**

Time spent: 3 hours

## Required Features

The following **required** functionality is completed:

- [x] The web app uses only HTML, CSS, and JavaScript without a frontend framework
- [x] The web app displays a title
- [x] The web app displays a list of items
- [x] Each item includes shared attributes such as name, category, price, description, and image
- [x] The app uses PicoCSS for styling
- [x] The app is served using an Express backend
